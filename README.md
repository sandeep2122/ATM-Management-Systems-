# 🏦 ATM Management System (Python)

## 📌 Project Description

This is a simple ATM Management System built using Python.
It allows users to perform basic banking operations like checking balance, depositing money, withdrawing money, and viewing transaction history.

---

## 🚀 Features

* 🔐 PIN Authentication (3 attempts)
* 💰 Check Balance
* ➕ Deposit Money
* ➖ Withdraw Money
* 📜 Transaction History (with Date & Time)

---

## 📁 Project Structure

```
atm/
│
├── main.py
├── balance.py
├── deposit.py
├── withdraw.py
├── transaction.py
└── pin.py
```

---

## ▶️ How to Run

1. Open terminal / command prompt
2. Navigate to project folder
3. Run the program:

```
python main.py
```
